# Resources

Links to official documentation, learning materials, and community resources for the tools in this stack.

## Claude

- [Claude](https://claude.ai/) — the chatbot interface
- [Claude Code](https://claude.ai/code) — installation page for Claude Code
- [Claude Documentation](https://docs.anthropic.com/) — official docs from Anthropic
- [Claude Code Documentation](https://docs.anthropic.com/en/docs/claude-code) — detailed Claude Code reference

## Obsidian

- [Obsidian](https://obsidian.md/) — download page
- [Obsidian Help](https://help.obsidian.md/) — official documentation
- [Give Me 15 Minutes. I'll Teach You 80% of Obsidian](https://www.youtube.com/watch?v=z4AbijUCoKU) — Nick Milo's beginner video (YouTube, 15 min)
- [Obsidian Community Forum](https://forum.obsidian.md/) — ask questions and browse discussions
- [Claude Sidebar Plugin](https://github.com/derek-larson14/obsidian-claude-sidebar) — the plugin that puts Claude Code in Obsidian's sidebar

## Git and GitHub

- [Git](https://git-scm.com/) — download page (needed for Windows; Mac uses Xcode tools)
- [GitHub](https://github.com/) — create an account
- [GitHub CLI](https://cli.github.com/) — the `gh` command-line tool
- [Git Documentation](https://git-scm.com/doc) — official reference (technical)
- [GitHub Docs: Getting Started](https://docs.github.com/en/get-started) — GitHub's beginner guides

## Markdown

- [Markdown Guide](https://www.markdownguide.org/) — comprehensive reference for Markdown syntax
- [Obsidian Markdown Reference](https://help.obsidian.md/Editing+and+formatting/Basic+formatting+syntax) — Obsidian-specific formatting

## Node.js

- [Node.js](https://nodejs.org/) — download page (required by Claude Code)

## Other Useful Tools

- [Homebrew](https://brew.sh/) — package manager for Mac (makes installing tools easier)
- [Rectangle](https://rectangleapp.com/) — free window manager for Mac (helpful when you have Obsidian and a browser open side by side)
