# Installing Obsidian

Obsidian is a free note-taking app that stores your notes as plain text (Markdown) files in a folder on your computer.

## Download and Install

1. Go to [obsidian.md](https://obsidian.md/)
2. Click the download button for your platform (Mac or Windows)
3. Run the installer

**Mac:** Open the downloaded `.dmg` file and drag Obsidian to your Applications folder.

**Windows:** Run the downloaded `.exe` installer and follow the prompts.

> [!tip]
> You don't need an Obsidian account. You may someday want to sign up with Obsidian — they're wonderful folks — but skip that for now and just download the software.

## First Launch

When you first open Obsidian, it will ask you to open or create a vault. For now, you can:

- **Create a new vault** to explore the interface, or
- **Wait** and come back to this after you've set up Git and GitHub (the [Your First Vault](Your%20First%20Vault.md) page covers creating a vault connected to GitHub)

If you create a vault to explore, remember to put it in a folder that is **not** synced by iCloud, OneDrive, or Dropbox (see [System Requirements](System%20Requirements.md) for recommended locations).

## Getting Oriented

Obsidian's interface has three main areas:

- **Left sidebar** — a file browser showing all the files in your vault
- **Center** — the editor where you read and write
- **Right sidebar** — where Claude Code will live once you set it up

You can open multiple files in tabs, like a web browser. Click a file in the left sidebar to open it.

That's really all you need to know to get started. You'll pick up more as you go.
