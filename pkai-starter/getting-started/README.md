# Getting Started with Claude Code, Obsidian, Git, and GitHub

Welcome! This wiki walks you through setting up a modern knowledge-work toolkit from scratch. No prior experience with terminals, Git, or code editors is required.

By the end, you'll have:

- **Claude Code** — an AI assistant that works directly with your files
- **Obsidian** — a powerful, free note-taking app that uses plain Markdown files
- **Git and GitHub** — version control and cloud backup for everything you write

These tools work together as an integrated system. You write in Obsidian, Claude Code helps you manage and sync your work, and GitHub keeps everything backed up and shareable.
