# Git for Non-Developers

A plain-language guide to Git and GitHub for people who have never used version control. Written for the PKAI Founders program.

## Who this is for

You've used Google Docs. You've collaborated on shared files. You've never opened a terminal on purpose. That's fine -- this guide assumes exactly that starting point.

You'll be using **Claude Code** as your interface to Git. That means you type English, and Claude Code handles the technical commands. You don't need to memorize anything.
