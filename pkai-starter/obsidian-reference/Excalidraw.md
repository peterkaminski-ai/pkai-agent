# Excalidraw

A community plugin that brings a full-featured whiteboard and drawing tool into Obsidian. Hand-drawn style by default, but supports precise shapes, text, images, embedded notes, and scripting.

**Install from:** Community plugin directory | [GitHub](https://github.com/zsviczian/obsidian-excalidraw-plugin)

## Installation

1. Go to **Settings > Community Plugins > Browse**
2. Search for **Excalidraw**
3. Click **Install**, then **Enable**

## What it does

- **Freeform drawing:** sketches, diagrams, flowcharts, wireframes, architecture maps
- **Embed notes:** drag vault notes onto the canvas; they stay linked and update live
- **Embed in notes:** use `![[drawing.excalidraw]]` to embed a drawing inside a markdown note
- **Image support:** paste or drag images onto the canvas
- **Libraries:** reusable shape libraries for icons, UI components, and more
- **LaTeX:** render math equations on the canvas
- **Scripting:** automate drawing operations with the Excalidraw scripting API

## When it's useful

- **Visual explanations:** draw a concept instead of describing it in text
- **Diagrams and flowcharts:** create process maps, system architectures, decision trees
- **Collaborative design:** share `.excalidraw` files in a shared vault for others to annotate
- **Storyboarding:** lay out a sequence of ideas or steps visually

## Excalidraw vs. Canvas

| | Canvas (core) | Excalidraw |
|---|---|---|
| **Focus** | Arranging existing notes spatially | Drawing and diagramming |
| **Cards** | Notes, text, web links | Shapes, text, images, embedded notes |
| **Drawing** | No freehand drawing | Full drawing tools |
| **Style** | Clean, structured | Hand-drawn by default (switchable) |
| **Best for** | Organizing notes | Creating visuals |

They complement each other. Use Canvas for spatial note organization, Excalidraw for creating visual content.
