# Resources

Links to official documentation, community resources, and tools.

## Official

- [Obsidian website](https://obsidian.md) -- download, pricing, and overview
- [Obsidian Help](https://help.obsidian.md) -- official documentation for all features and settings
- [Obsidian Releases](https://obsidian.md/changelog) -- what's new in each version
- [Obsidian Community Plugins](https://obsidian.md/plugins) -- browse the plugin directory

## Community

- [Obsidian Forum](https://forum.obsidian.md) -- official community forum for questions, feature requests, and showcases
- [Obsidian Discord](https://discord.gg/obsidianmd) -- real-time chat with the community and developers
- [r/ObsidianMD](https://reddit.com/r/ObsidianMD) -- Reddit community

## Plugin repositories

- [BRAT](https://github.com/TfTHacker/obsidian42-brat) -- install beta plugins from GitHub
- [Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin) -- whiteboard and drawing
- [Templater](https://github.com/SilentVoid13/Templater) -- advanced template scripting
- [Strange New Worlds](https://github.com/TfTHacker/obsidian42-strange-new-worlds) -- inline reference counters
- [Folder Notes](https://github.com/LostPaul/obsidian-folder-notes) -- turn notes into folders
- [Markmind](https://github.com/MarkMindCkm/obsidian-markmind) -- mind mapping and outlining
- [Obsidian Mind Map](https://github.com/lynchjames/obsidian-mind-map) -- heading-based mind maps
- [Claude Sidebar Plugin](https://github.com/derek-larson14/obsidian-claude-sidebar) -- Claude Code in the sidebar

## Related tools

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) -- the CLI tool that powers the Claude Sidebar Plugin
- [GitHub CLI (gh)](https://cli.github.com) -- command-line tool for GitHub, useful for creating repos from Obsidian via Claude Code
- [Excalidraw web](https://excalidraw.com) -- the standalone web version of Excalidraw (also available as an Obsidian plugin)

## Learning

- [Obsidian Help: Plugins](https://help.obsidian.md/Plugins) -- official guide to core plugins
- [Obsidian Help: Properties](https://help.obsidian.md/Editing+and+formatting/Properties) -- frontmatter documentation
- [Obsidian Help: Search](https://help.obsidian.md/Plugins/Search) -- search operators reference
