# What Is Obsidian

Obsidian is a note-taking application that stores everything as plain Markdown files in a folder on your computer. There's no proprietary database, no cloud lock-in, and no account required. Your notes are text files you can read with any editor, back up with Git, and keep forever.

## What makes it different

**Local-first.** Your notes live on your filesystem, not on someone else's server. You choose how and whether to sync them.

**Wiki-style linking.** You connect notes with `[[wiki-links]]`. Type `[[` and start typing a note name -- Obsidian autocompletes it. These links create a web of connections that you can visualize, search, and navigate.

**Plugin ecosystem.** Obsidian ships with a set of core plugins (search, backlinks, templates, graph view, and more). Beyond that, there are hundreds of community plugins for everything from mind mapping to AI integration to Git-based version control.

**Extensible but grounded.** At its core, Obsidian is a Markdown editor. The plugins add capabilities without changing the underlying format. If you stopped using Obsidian tomorrow, your notes would still be perfectly readable text files.

## What it's good for

- **Personal knowledge bases** -- a place for notes, research, project plans, and reference material
- **Collaborative wikis** -- shared vaults with Git for version control
- **AI-assisted workflows** -- Claude Code runs inside Obsidian via a sidebar plugin, giving you an AI assistant that can read and edit your notes directly
- **Structured and unstructured notes** -- plain text for quick capture, frontmatter properties for structured metadata, Bases for queryable views

## What it's not

Obsidian is not a word processor, a project management tool, or a database. It's a thinking environment. It works best when you treat it as a place to write, connect, and organize ideas -- not as a replacement for specialized tools.

## Getting it

Download from [obsidian.md](https://obsidian.md). Available for Mac, Windows, Linux, iOS, and Android. You do not need an account on the Obsidian website to use the desktop app.
