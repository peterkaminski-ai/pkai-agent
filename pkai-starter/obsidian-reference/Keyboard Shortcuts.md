# Keyboard Shortcuts

Essential Obsidian keyboard shortcuts. All shortcuts use **Cmd** on Mac and **Ctrl** on Windows/Linux.

## Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd + P | Open Command Palette |
| Ctrl/Cmd + O | Quick switcher (open a note by name) |
| Ctrl/Cmd + Shift + F | Search across vault |
| Alt + left arrow | Navigate back |
| Alt + right arrow | Navigate forward |
| Ctrl/Cmd + click | Open link in new tab |

## Editing

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd + B | Bold |
| Ctrl/Cmd + I | Italic |
| Ctrl/Cmd + K | Insert link |
| Ctrl/Cmd + ] | Indent |
| Ctrl/Cmd + [ | Unindent |
| Ctrl/Cmd + Enter | Toggle checkbox |
| Ctrl/Cmd + D | Delete current line |
| Ctrl/Cmd + Z | Undo |
| Ctrl/Cmd + Shift + Z | Redo |

## Formatting

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd + 1-6 | Apply heading level 1-6 |
| Ctrl/Cmd + Shift + ] | Toggle bulleted list |
| Ctrl/Cmd + Shift + [ | Toggle numbered list |

## Panes and Layout

| Shortcut | Action |
|----------|--------|
| Ctrl/Cmd + \ | Toggle left sidebar |
| Ctrl/Cmd + Shift + \ | Toggle right sidebar |
| Ctrl/Cmd + W | Close current tab |
| Ctrl/Cmd + N | New note |

## Customizing shortcuts

You can reassign any shortcut and add new ones:

1. Go to **Settings > Hotkeys**
2. Search for the command you want to bind
3. Click the plus icon and press your desired key combination

Every command in Obsidian (including those from community plugins) can have a hotkey. If you find yourself reaching for the Command Palette for the same action repeatedly, give it a shortcut.

## The most important shortcut

**Ctrl/Cmd + P** (Command Palette) is the one shortcut to memorize if you memorize nothing else. It gives you access to every command in Obsidian. Start typing what you want to do, and it'll find it.
