# Essential Community Plugins

A curated list of community plugins worth knowing about. These are installed via **Settings > Community Plugins > Browse** unless noted otherwise.

## Must-have

### BRAT
**Install from:** Community plugin directory

BRAT (Beta Reviewers Auto-update Tester) lets you install plugins directly from GitHub before they're published to the community directory. You need this to install the Claude Sidebar Plugin and other beta plugins. See [Installing Community Plugins](Installing%20Community%20Plugins.md) for setup details.

### Claude Sidebar Plugin
**Install from:** BRAT (repo: `derek-larson14/obsidian-claude-sidebar`)

Docks Claude Code into the Obsidian sidebar as a JavaScript terminal emulator. You interact with it like a terminal, not a GUI. See [Claude Sidebar Plugin](Claude%20Sidebar%20Plugin.md) for full details.

## Visual and Spatial

### Excalidraw
**Install from:** Community plugin directory | [GitHub](https://github.com/zsviczian/obsidian-excalidraw-plugin)

A full-featured whiteboard and drawing tool inside Obsidian. Freeform sketching, diagrams, flowcharts, embedded notes, image support, and even scripting. Hand-drawn style by default, switchable to precise shapes. See [Excalidraw](Excalidraw.md).

### Markmind
**Install from:** Community plugin directory | [GitHub](https://github.com/MarkMindCkm/obsidian-markmind)

Full-featured mind mapping and outlining. Two modes: **Basic** (markdown-based, edits sync between map and text) and **Rich** (freeform visual editor with drag-and-drop, colors, images). Good for planning, brainstorming, and visual outlining.

### Obsidian Mind Map
**Install from:** Community plugin directory | [GitHub](https://github.com/lynchjames/obsidian-mind-map)

Renders any note as a read-only mind map from its heading structure. Each heading becomes a node; nesting creates branches. Useful for reviewing the structure of long notes at a glance. Lighter than Markmind -- no editing, just visualization.

## Organization

### Folder Notes
**Install from:** Community plugin directory | [GitHub](https://github.com/LostPaul/obsidian-folder-notes)

Turn any note into a folder in one step. Run **Folder Notes: Create Folder Note** from the Command Palette. This creates a folder with the same name and moves the note inside it as the index page. Useful when a note grows and needs sub-notes.

### Strange New Worlds
**Install from:** Community plugin directory | [GitHub](https://github.com/TfTHacker/obsidian42-strange-new-worlds)

Shows inline counters next to every `[[wiki-link]]` and `#tag`, telling you how many other notes reference the same target. Click the badge to see the list of referencing notes. Makes the network visible without opening the graph or the backlinks panel.

## Advanced

### Templater
**Install from:** Community plugin directory | [GitHub](https://github.com/SilentVoid13/Templater)

A more powerful alternative to the core Templates plugin. Adds date arithmetic, file metadata access, conditionals, user prompts, and automatic templates per folder. See [Templates and Templater](Templates%20and%20Templater.md) for a comparison.
